# Agent Rules

## Execution Mode
- Default mode: FAST
- Prioritize concise responses.
- Avoid long explanations unless explicitly requested.

## Code Safety
- DO NOT automatically modify existing files.
- DO NOT refactor code unless explicitly asked.
- DO NOT create, edit, or delete files without user confirmation.
- Only suggest code changes in chat.
- Wait for explicit approval before applying patches.

## Response Style
- Provide direct answers.
- Focus on actionable output.
- No unnecessary commentary.
